# Async page data loader pattern in React with React Router and Tanstack Query

This repository contains an example for the blog post [Async page data loader pattern in React with React Router and Tanstack Query](https://gregkedzierski.com/essays/async-page-data-loader-pattern-in-react-with-react-router-and-tanstack-query/).

## Set up

Install dependencies:

```bash
yarn
```

Start the development server:

```bash
yarn dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
